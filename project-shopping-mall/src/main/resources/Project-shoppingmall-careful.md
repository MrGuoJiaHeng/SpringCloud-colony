<h1>Spring熔断器</h1>
<div>
    <p>
        在需要熔断的方法前加上注解 “ @HystrixCommand(fallbackMethod = "回调方法") ”
        <br/>
        其中回调方法为当拥有该注解的方法发出的请求超过设定时间仍未响应，或多次调用均报错时，会转发到另一个方法。
        <br/>
        回调方法填写的就是要转发到哪一个方法做处理
        <br/>
        转发的方法需满足以下条件:
        <br/>
        <1 方法的返回值和原方法一致
        <br/>
        <2 方法的参数与原方法一致
        <br/>
        若不满足上述条件，此回路方法无法被识别。
    </p>
</div>
